📋 Pertanyaan Umum (FAQ)

Bagaimana cara memulai chat? ▼

Cukup ketik pertanyaan di kolom chat lalu tekan Enter atau klik tombol Kirim. AI akan merespon dengan jawaban dari database manual kami yang berisi 2000+ respons!

Apakah butuh API key / internet? ▼

TIDAK! AI ini 100% offline dan tidak membutuhkan API key sama sekali. Semua respons ditulis manual oleh LifxXcodetz dan berjalan di browser Anda.

Bisa pakai singkatan? ▼

BISA! AI akan mendeteksi singkatan umum:
• "hlo", "helo" → "halo"
• "mksh", "thx" → "terima kasih"
• "mf", "maap" → "maaf"
• "gpp" → "tidak apa apa"
Dan masih banyak lagi!

Siapa pembuat AI ini? ▼

AI ini dibuat oleh LifxXcodetz, seorang developer ganteng yang ingin menyediakan asisten AI gratis tanpa API key untuk masyarakat Indonesia. Beliau juga yang membuat semua 2000+ respons manual ini!

📖 Panduan Penggunaan

1

Memulai Chat

Buka halaman Dashboard, ketik pertanyaan di kolom chat, tekan Enter atau klik Kirim. AI akan merespon dengan jawaban dari database.

2

Menjelajahi Kategori

Klik menu "Kategori" di sidebar untuk melihat semua 26 kategori. Klik kategori untuk melihat pertanyaan-pertanyaan populer di dalamnya.

3

Melihat Statistik

Di halaman Statistik, Anda bisa melihat grafik aktivitas, pertanyaan terpopuler, dan analisis penggunaan AI.

4

Mengatur Preferensi

Di halaman Pengaturan, Anda bisa mengubah tema, bahasa, notifikasi, dan berbagai preferensi lainnya.

5

Export Chat

Klik ikon 📥 di chat header untuk mendownload seluruh percakapan sebagai file teks.
