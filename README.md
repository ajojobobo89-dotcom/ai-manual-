## 📋 Deskripsi

**AI Asisten Manual** adalah asisten virtual berbasis web yang **100% GRATIS** dan **TANPA API KEY**. Semua respons ditulis manual oleh developer, sehingga tidak memerlukan koneksi internet ke server eksternal.

Dibuat dengan ❤️ oleh **LifxXcodetz** (developer ganteng) untuk masyarakat Indonesia.

## ✨ Fitur Unggulan

- ✅ **2000+ Respons Manual** - Database lengkap berbagai topik
- ✅ **26 Kategori** - Dari teknologi hingga motivasi
- ✅ **Deteksi Singkatan** - "hlo" otomatis dibaca "halo"
- ✅ **100% Offline** - Tidak perlu internet setelah dimuat
- ✅ **Tanpa API Key** - Benar-benar gratis selamanya
- ✅ **Multi Halaman** - Dashboard, kategori, statistik, pengaturan, bantuan
- ✅ **Export Chat** - Simpan percakapan ke file
- ✅ **Dark/Light Mode** - Tema bisa diganti
- ✅ **Responsive Design** - Bisa diakses dari HP/tablet/desktop
